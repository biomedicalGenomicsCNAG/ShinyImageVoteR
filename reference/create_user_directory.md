# Create user directory structure

Creates the directory structure for a user under their institute folder.
This ensures both the institute and user directories exist.

## Usage

``` r
create_user_directory(user_data_dir, institute, user_id)
```

## Arguments

- user_data_dir:

  Character. Base directory for user data

- institute:

  Character. Institute name

- user_id:

  Character. User ID

## Value

Character. Path to the created user directory
