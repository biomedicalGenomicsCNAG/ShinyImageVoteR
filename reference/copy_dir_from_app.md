# Copy a directory from the app to a target path This function copies a directory from the app's default environment to a specified target path.

Copy a directory from the app to a target path This function copies a
directory from the app's default environment to a specified target path.

## Usage

``` r
copy_dir_from_app(target_dir_path)
```

## Arguments

- target_dir_path:

  Character. Path to the target directory where the app directory will
  be copied.

## Value

Character path to the copied directory
