# User statistics module UI

Provides the user interface for displaying statistics related to the
current user's voting activity within the B1MG Variant Voting app. This
typically includes a table of vote counts or annotations per session,
along with a refresh button.

## Usage

``` r
userstatsUI(id, cfg)
```

## Arguments

- id:

  A string identifier for the module namespace.

## Value

A Shiny UI element (`fluidPage`) for rendering user statistics.
