# ShinyImgVoteR

## Introduction

## Purpose and Scope

- **Purpose**: Collaborative image review and voting

- **Delivered in the form of?** Shiny web application R package

- **Application Purpose**: Display a set of images one at a time to
  users. Capture structured votes (agree/disagree) with standardized
  options.

- **Key Features**:

  1.  Users can be grouped and need to login to access the application.
  2.  Hotkeys allow quick voting via keyboard shortcuts.
  3.  Possibility to go back to previous images to change votes.
  4.  The votes of each users are stored in a TSV for easy downstream
      analysis.
  5.  A leaderboard displays userâ€™s groups voting progress in real-time.
  6.  Users can learn about their voting behavior (time till vote is
      casted, images voted per session etc.) via personal statistics.
